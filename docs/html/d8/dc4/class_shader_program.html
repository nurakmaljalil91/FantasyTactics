Detailed Description

Class for managing OpenGL shaders.

This class handles loading, compiling, and linking vertex and fragment shaders. It also provides methods to set uniform variables in the shader program.

ShaderProgram::ShaderProgram ( const char * vsFilename,

Constructor that loads shaders from specified files.

Parameters
vsFilenamefilename for the vertex shader.
fsFilenamefilename for the fragment shader.

This constructor attempts to load and compile the vertex and fragment shaders from the given files. If successful, it links them into a shader program. If the shaders cannot be loaded, the shader handle will remain 0.
