Detailed Description

Represents a 3D mesh loaded from an OBJ file.

This class handles loading a mesh from a Wavefront OBJ file and rendering it using OpenGL. It supports basic vertex attributes like position, normal, and texture coordinates.

bool Mesh::loadObj ( const std::string & filename)

Loads a Wavefront OBJ model from a file.

Parameters
filename
Returns
if the model was loaded successfully.

This function reads the OBJ file, extracts vertex positions, normals, and texture coordinates, and initializes OpenGL buffers for rendering. NOTE! This is not a complete, full-featured OBJ loader. It is greatly simplified. Assumptions:

  • an OBJ file must contain only triangles.
  • We ignore materials.
  • We ignore normals.
  • Only commands "v", "vt", and "f" are supported.
