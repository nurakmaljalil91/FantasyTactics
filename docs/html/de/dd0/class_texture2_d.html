Detailed Description

This class represents a 2D texture in OpenGL.

The Texture2D class is responsible for loading, binding, and managing 2D textures in OpenGL. It uses the stb_image library to load images from files and creates OpenGL texture objects. The class provides methods to load textures from files, bind and unbind textures, and manage texture parameters. It is designed to be used in a graphics rendering context where OpenGL is used for rendering 2D and 3D graphics.
